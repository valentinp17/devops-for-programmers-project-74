It's a ci-cd for JavaScript application copying from repo [JS Fastify Blog](https://github.com/hexlet-components/js-fastify-blog)

### Requirements
- docker
- docker compose V2 
- ruby >= 3.0.0 
- make

### Before everything run

```make app-setup:```

### Run tests

```make app-test```

### start dev server: 

```make dev```
